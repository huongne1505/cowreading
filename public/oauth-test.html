Cấu hình cần thiết:

  1. Tạo Facebook App tại: Facebook Developers
  2. Tạo Google OAuth Client tại: Google Cloud Console
  3. Cập nhật file .env với Client ID và Secret
  4. Redirect URIs phải match với cấu hình OAuth

Chú ý:

OAuth sẽ hoạt động khi bạn đã cấu hình đúng Client ID và Secret từ Facebook và Google. Hiện tại đang sử dụng giá trị placeholder trong file .env.
